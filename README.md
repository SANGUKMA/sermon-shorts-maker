# 설교쇼츠제작에이전트

설교 원고를 입력하면 약 1분 30초 분량의 유튜브 쇼츠 영상(9:16)을 자동으로 만들어 드립니다.

## 무엇을 하나요?

1. **설교 원고**를 텍스트로 붙여넣기
2. **TTS** — 목사님 음성 클론으로 나레이션 생성
3. **Imagen 4** — 설교 내용에 맞는 일러스트 10장 생성
4. **Kling AI** — 일러스트를 움직이는 영상으로 변환 (각 10초)
5. **합성** — 9:16 캔버스에 상단 제목 / 1:1 영상 / 하단 교회명 합성
6. **다운로드** — 완성된 mp4 파일과 메타데이터 (제목/설명/태그) 제공

## 사용법

1. 브라우저로 앱 URL 접속
2. **설교 제목**, **성경 본문**, **교회명** 입력
3. **설교 원고**를 텍스트 박스에 붙여넣기 (긴 원고는 핵심 부분만 약 1000자 정도)
4. **생성하기** 버튼 클릭
5. 약 **15~20분** 대기 (탭을 닫으면 안 됩니다 — 진행 상황이 화면에 표시됩니다)
6. 완성되면 **mp4 다운로드** + **메타데이터 복사**
7. YouTube Studio에 직접 영상 업로드 + 메타데이터 붙여넣기

## 주의사항

- **생성 중 탭 닫지 마세요** — 진행 상황이 사라집니다
- **약 15~20분 소요** — 영상 길이가 길어서 처리 시간이 깁니다
- **API 비용** — 1개 영상 약 $5~6 (목사님 API 키 사용)
- **YouTube 업로드는 직접** — 영상이 만들어지면 다운로드해서 YouTube Studio에서 직접 업로드하세요

## 문제가 생기면?

- "쇼츠 만들기" 클릭 후 화면이 멈춘 것 같으면 — **그대로 두세요**, Kling이 한 클립 만드는 데 1분 정도 걸립니다
- 에러 메시지가 뜨면 — **개발자에게 스크린샷 공유**
